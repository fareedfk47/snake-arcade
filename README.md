# 🐍 Snake Arcade

A retro-inspired Snake Arcade game built with **HTML5, CSS3, and Vanilla JavaScript (ES6)**. This project recreates the classic Snake experience with a modern CRT terminal-inspired interface while demonstrating clean code architecture, responsive design, collision detection, state management, and dynamic difficulty progression.

## ✨ Features

- 🎮 Classic Snake gameplay
- ⚡ Dynamic speed progression
- 🏆 Persistent high score using Local Storage
- ⏸️ Pause and Resume support
- 🎯 Random fruit generation
- 💥 Collision detection
- ⌛ Real-time game timer
- 📱 Responsive game board
- 🖥️ CRT terminal-inspired interface
- 🚀 Smooth game loop
- 🔄 Restart without refreshing the page

## 🎮 Controls

| Key | Action |
|------|--------|
| ⬆️ | Move Up |
| ⬇️ | Move Down |
| ⬅️ | Move Left |
| ➡️ | Move Right |
| **Space** | Start / Pause / Resume |
| **Enter** | Start / Restart |

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/fareedfk47/snake-arcade.git
```

Navigate to the project folder

```bash
cd snake-arcade
```

Run the project

Simply open `index.html` in your browser or use **VS Code Live Server**.
